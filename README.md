## SCU_Mail有哪些注意事项？

* 本项目支持`群发邮件`，支持从通讯录中导入数据，注意：！！不要更改通讯录结构，没有做防错处理。
    *  `通讯录结构`
    *  姓名
    *  联系地址
    *  邮件主题
    *  邮件正文
    *  附件
    *  `其中附件列可为空，其他列必填！`
* 默认发群邮件时候最多使用10个线程，如有需要可以人为更改代码
* 发送群邮件时候，会`生成log日志`，分为两个，一个是成功发送的log日志，一个人失败的log日志，包括失败原因
* 如有需要，源码一同分发，可以`自行修改代码`
